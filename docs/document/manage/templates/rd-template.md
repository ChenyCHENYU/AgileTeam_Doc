# RD 前端工程师模板库

## 说明

> 定义模板库的目的是给前端工程师一个`可参考`，可具象的一个输出`动作实例`，它是一个约定俗成的`标准产物`，基于实际场景需要的一个相对 "`最佳实践`"，它应该是`可迭代`和`可持续集成`的。

## 模板

对于前端开发的模板，希望从内部小组制定一套，前端工程师角色的要求以及标准化的输出。
